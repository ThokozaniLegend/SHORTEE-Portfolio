About SHORTEE

SHORTEE was inspired by the need for a simple yet powerful URL shortener that empowers users to not only shorten links but track their performance. As someone who frequently shares links online, I needed a tool that provided in-depth insights without the complexity of enterprise tools.

This project was developed as part of my **Portfolio Project** at ALX INSTITUTE. It highlights my skills in full-stack web development, showcasing both front-end and back-end technologies. Feel free to explore the code and features below!
